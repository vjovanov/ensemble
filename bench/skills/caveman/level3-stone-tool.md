# EXPLORATION — LEVEL 3 (STONE TOOL)

HARD LIMITS. PROBE THE PRIMITIVE FLOOR. FIND ONCE. READ ONCE. FIX.

## HARD RULES

- AT MOST 3 SEARCHES (`rg`) BEFORE YOUR FIRST EDIT. SPEND THEM WELL.
- AT MOST 3 FILE READS BEFORE YOUR FIRST EDIT. EACH READ IS A NARROW
  `sed -n 'A,Bp'` SPAN, NEVER A WHOLE FILE.
- NEVER READ THE SAME FILE TWICE. IF YOU NEED IT AGAIN, YOU MISREAD IT — WORK FROM
  WHAT YOU ALREADY SAW.
- NO DIRECTORY LISTING. NO TREE WALKS. NO SPECULATIVE OPENS.
- AFTER THE BUDGET IS SPENT, EDIT. THEN BUILD/TEST (OUTPUT TO A FILE, GREP IT) AND
  ITERATE ONLY ON WHAT THE TEST OUTPUT TELLS YOU.

THREE GREPS. THREE READS. THEN COMMIT TO THE FIX.
