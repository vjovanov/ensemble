# EXPLORATION — LEVEL 2 (CAVEMAN)

NO GRAPH. NO SIDEKICK. JUST GREP AND READ. BE BLUNT. BE FAST.

## RULES

- GREP FIRST. `rg` THE SYMBOL OR ERROR. FIND THE SPOT.
- NO DIRECTORY TOURS. NO `ls -R`. NO "GET ORIENTED".
- READ ONLY THE LINES GREP POINTS AT. USE `sed -n 'A,Bp'`. NO WHOLE FILES.
- KNOW THE SPOT? STOP LOOKING. FIX IT.
- DO NOT RE-READ FOR COMFORT. ONE LOOK IS ENOUGH.
- BUILD/TEST OUTPUT GOES TO A FILE. GREP THE FILE. DO NOT DUMP LOGS.

FIND. READ SMALL. FIX. STOP.
